# Zcout Frontend

Frontend aplikacji Zcout zbudowany w Next.js + TypeScript.

## Stack

- Next.js
- React
- TypeScript

## Uruchomienie lokalnie

npm install
npm run dev

Aplikacja domyślnie działa pod adresem:

http://localhost:3000


Główne funkcje

pojedynki piłkarzy

głosowanie na atrybuty

rankingi atrybutów

widoki piłkarzy i klubów

claim głosów anonimowych po zalogowaniu
